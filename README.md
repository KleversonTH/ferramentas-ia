# Ferramentas IA — Analisador de Produtos Mercado Livre

Plataforma web com autenticação e banco de dados para análise de produtos com alta demanda e margem de lucro no Mercado Livre, utilizando Inteligência Artificial.

## 🚀 Funcionalidades

- Análise de produtos com IA por categoria
- Tendência de mercado (crescendo, estável, caindo)
- Links diretos para fornecedores (AliExpress, Shopee, Alibaba, ML)
- Sistema de favoritos
- Autenticação com JWT
- Painel admin para gerenciar usuários
- Banco de dados PostgreSQL

## 🛠️ Tecnologias

- **Backend:** Node.js, Express
- **Banco de dados:** PostgreSQL
- **Autenticação:** JWT
- **IA:** OpenRouter API (Google Gemma)
- **Frontend:** HTML, CSS, JavaScript
- **Deploy:** Railway

## 📦 Como rodar localmente

```bash
npm install
node server.js
```
